# Math Solver App

A Rocket.Chat app generated by the AI Rocket.Chat Apps Generator. This app demonstrates a complete slash command implementation that evaluates math expressions from chat.

## Problem Statement

Users want to quickly evaluate expressions without leaving Rocket.Chat. This app provides a `/math` slash command that calculates results via a public API.

## Implemented Features

- **Slash Command:** `/math <expression>` - evaluates expressions typed in chat
- **Error Handling:** Handles invalid expressions and API failures
- **Public API:** Uses mathjs.org (no API key required)
- **Message Patterns:**
  - Public result message in the room
  - Ephemeral error message only to the requesting user

## How to Test

1. Deploy the app to your Rocket.Chat instance.
2. In any chat, run: `/math 5 * (2 + 3)`
3. Expected output format:
   - `Math Expression: 5 * (2 + 3)`
   - `Result: 25`

## Example Commands

```
/math 4+5
/math 5+6
/math 5+12
/math 12+(6*5)/12
```

## Code Architecture

- **App Class:** `MathSolverAppApp` (extends `App` and implements `getName()`)
- **Feature Class:** `MathSlashCommand` (implements `ISlashCommand`)
- **API Pattern:** Math.js endpoint with encoded expression
- **Error Handling:** try/catch around HTTP calls + status code checks
